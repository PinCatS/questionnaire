# Финансовый опросник

Пошаговый финансовый опросник из 5 вопросов (тестовое задание)

## Технологии

- Angular 11
- Redux

## Как запустить

- Скачать или клонировать репозиторий на локальный компьютер
- Выполнить команды npm i && ng serve
